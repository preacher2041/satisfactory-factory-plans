Node placement is by dependency depth, not build floor — Heavy Modular Frame sits one tier up because every ingredient it needs comes straight from storage, while Automated Wiring and Computer each need one assembler's output first. Copper Sheet and Plastic are each split by destination the same way Steel Pipe and Cable are: the Circuit-Board-bound portion (32+20 Copper Sheet, 64+40 Plastic) is separate from Plastic's other flow straight into Computer (64/min, unrelated to Circuit Board despite the matching number). Circuit Board itself is split the same way one tier up — 16/min feeding Computer, 10/min feeding the ACU directly. Totals are each node's rate × 50 minutes, the time this factory takes to clear a 100-unit ACU batch at its 2/min target. Every Assembler/Manufacturer node also shows the clock speed and power shard count its output rate requires — Heavy Modular Frame is the only one that hits its target at stock 100%; the other seven all need overclocking, 14 shards combined.
