From Industrial Storage to Adaptive Control Unit

One node per physical machine, one storage node per machine it feeds — no node splits its output across more than one machine above it unless that machine is itself a single unit receiving an even split (Stator's one machine feeding two Automated Wiring machines, 5/min each, is the only case). Rates are per minute at this factory's 2 ACU/min target; totals are each node's own rate × 50 minutes, the time to clear a 100-unit batch.

Circuit Board is now 4 separate machines split into two dedicated 2-machine banks by destination, not 2 machines sharing a pool that gets divided after the fact — Circuit Board #1/#2 run at 106.67% and feed only Computer, #3/#4 run at 66.67% (no shard needed, an underclock) and feed only the ACU. That redesign drops Circuit Board's shard cost from 4 to 2. Automated Wiring is likewise 2 separate 200% machines rather than one combined node. Stator remains the one exception to "storage feeds exactly one machine": it's a single machine whose 10/min output splits evenly in half to the two Automated Wiring machines — a legal 50/50 split, and Stator itself isn't a storage-fed leaf node, so the bottom-row rule doesn't apply to it. Full shard total: 2 (Stator) + 2 (Circuit Board #1/#2) + 0 (Circuit Board #3/#4) + 0 (HMF) + 2+2 (Automated Wiring #1/#2) + 2 (Computer) + 2 (ACU) = 12 shards.
